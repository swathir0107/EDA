📌 Project Title
Customer Churn Analysis and Prediction using Python
This project focuses on analyzing customer behavior data to identify which customers are likely to leave a company (churn), helping businesses improve customer retention. 
📖 About the Project

Customer churn means when customers stop using a company’s service. In this notebook, customer data such as usage patterns, charges, contract type, and demographics are analyzed to understand churn behavior.
The project performs:
Dataset loading
Data cleaning
Missing value handling
Encoding categorical columns
Exploratory Data Analysis (EDA)
Churn pattern visualization
Feature selection
Train/Test preparation
Data ready for machine learning models
This type of project is common in telecom, banking, insurance, and subscription businesses.
